# MicroFrontEndsExample
Example implementing a React Micro Frontend into an Angular Shell using Module Federation

# How to Start
##### Main App
 
> cd ./shell
> ng serve

Navigate to `http://localhost:4200/`.

##### MFE

> cd ./mfe
> npm start

Navigate to `http://localhost:4201/`
